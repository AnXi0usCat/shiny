######################
ClsuterApp (Shiny App)
######################

This is a simple application allowes user to see a range of dendogram plots 
which are available in R to visualize the results of Hierarchical Clustering Algorithm. 
The Data used is mtcars and available in R datasets package.

In order to use the app just select a different dendogram from "Select Plot" select box.

in order to run the app locally the "shiny" and "ape" packages are required.
Both packages can be downloaded from CRAN:
